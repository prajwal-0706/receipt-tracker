Database schema (column names matter — use these EXACTLY):
  receipts(
    id            uuid PRIMARY KEY,         -- the receipts PK is called `id`, NOT `receipt_id`
    image_hash    text,
    image_path    text,
    merchant      text,
    receipt_date  date,

    confidence    real,
    created_at    timestamptz

    id           uuid PRIMARY KEY,
    receipt_id   uuid REFERENCES receipts(id),  -- `receipt_id` lives on line_items only

    quantity     numeric,
    unit_price   numeric,
    total_price  numeric,                       -- per-line total = unit_price * quantity
    category     text,
    category_confidence real
  )

Categories used in line_items.category:

Rules:
  - Output ONLY the SQL. No explanation, no markdown fences, no semicolons at the end.
  - Use only SELECT. Never UPDATE/INSERT/DELETE/DROP/ALTER.
  - Use only the tables `receipts` and `line_items`.
  - For "spent on X" questions, sum line_items.total_price WHERE category = 'X'.
  - For "biggest bill" questions, treat a bill as one receipt (use receipts.total, not item totals).
  - For relative dates like "last month", "this year", compute the range from {TODAY}.
  - For ANY merchant name in the question, use ILIKE with wildcards (e.g. merchant ILIKE '%techwave%'), never exact equality. Merchant names in receipts may differ in casing or trailing characters.
  - "Recent" alone (no specific timeframe) means the last 12 months, not 30 days — be lenient.
  - Every SELECT MUST have a FROM clause referencing `receipts` or `line_items`. Never write `SELECT COUNT(*)` or `SELECT SUM(...)` without a FROM — the result will be wrong (returns 1 row of constants).
  - When summing receipt totals, ALWAYS include `currency` in the SELECT and GROUP BY so the answer can mention the currency.
  - "How much did I spend at <merchant>" sums `receipts.total` filtered by merchant ILIKE — DO NOT add a category filter, do not query line_items, do not assume a category.
  - "Biggest/largest receipt" or "biggest bill" sums on `receipts.total`, selects `id`, `merchant`, `receipt_date`, `total`, `currency` — never `receipt_id` (that column does not exist on receipts).
  - The `category` column exists ONLY on `line_items`. It does NOT exist on `receipts`. For ANY question about spending in a category ("what did I spend on dining", "how much on groceries", etc.), the query MUST be `FROM line_items WHERE category = '...'` and sum `total_price` — never `FROM receipts WHERE category = '...'` (that will error).
  - If the question is unanswerable from this schema, return: SELECT 'unanswerable' AS error

Examples:

Q: How much did I spend on coffee last month?
SQL: SELECT COALESCE(SUM(li.total_price), 0) AS total_spent FROM line_items li JOIN receipts r ON r.id = li.receipt_id WHERE li.category = 'dining' AND (li.description ILIKE '%coffee%' OR li.description ILIKE '%latte%' OR li.description ILIKE '%espresso%') AND r.receipt_date >= date_trunc('month', current_date - interval '1 month') AND r.receipt_date < date_trunc('month', current_date)

Q: What was my biggest grocery bill in March?
SQL: SELECT r.id, r.merchant, r.receipt_date, r.total FROM receipts r WHERE r.id IN (SELECT receipt_id FROM line_items WHERE category = 'groceries') AND extract(month from r.receipt_date) = 3 AND extract(year from r.receipt_date) = extract(year from current_date) ORDER BY r.total DESC NULLS LAST LIMIT 1

Q: How much did I spend in total this year?
SQL: SELECT COALESCE(SUM(total), 0) AS total_spent, currency FROM receipts WHERE receipt_date >= date_trunc('year', current_date) GROUP BY currency

Q: How much did I spend in total?
SQL: SELECT COALESCE(SUM(total), 0) AS total_spent, currency FROM receipts GROUP BY currency

Q: What's my biggest receipt?
SQL: SELECT id, merchant, receipt_date, total, currency FROM receipts ORDER BY total DESC NULLS LAST LIMIT 1

Q: How much did I spend at TechWave Solutions?
SQL: SELECT COALESCE(SUM(total), 0) AS total_spent, COUNT(*) AS receipt_count, currency FROM receipts WHERE merchant ILIKE '%techwave%' GROUP BY currency

Q: Show my top 5 merchants by spend.
SQL: SELECT merchant, SUM(total) AS spent FROM receipts WHERE merchant IS NOT NULL GROUP BY merchant ORDER BY spent DESC NULLS LAST LIMIT 5

Q: How many receipts have I uploaded?
SQL: SELECT COUNT(*) AS receipt_count FROM receipts

Q: What's my average dining bill?
SQL: SELECT AVG(r.total) AS avg_bill FROM receipts r WHERE r.id IN (SELECT receipt_id FROM line_items WHERE category = 'dining')

Q: Tell me more about my recent transaction with TechWave Solutions
SQL: SELECT r.id, r.merchant, r.receipt_date, r.total, r.currency, li.description, li.quantity, li.total_price, li.category FROM receipts r LEFT JOIN line_items li ON li.receipt_id = r.id WHERE r.merchant ILIKE '%techwave%' ORDER BY r.receipt_date DESC LIMIT 50

Q: How much have I spent at Walmart?
